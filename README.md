# 프로젝트 명 : FreshKeeper

## 프로젝트 소개

FreshKeeper는 냉장고 속 식재료의 유통기한을 관리하고, 유통기한이 임박한 재료를 우선적으로 활용할 수 있도록 메뉴를 추천하는 Java 콘솔 프로그램이다.

사용자는 식재료명, 수량, 보관 위치, 유통기한을 입력할 수 있으며, 프로그램은 현재 날짜를 기준으로 남은 유통기한을 계산한다. 또한 보유 중인 식재료와 레시피 데이터를 비교하여 만들 수 있는 메뉴를 추천하고, 부족한 재료를 함께 표시한다.

## 프로젝트 목적

본 프로젝트는 냉장고 속 식재료의 유통기한을 효율적으로 관리하고, 음식물 낭비를 줄이는 것을 목적으로 한다.

기존에는 사용자가 냉장고를 직접 확인하거나 메모장에 식재료를 기록해야 했지만, 이러한 방식은 유통기한이 얼마나 남았는지 한눈에 파악하기 어렵다. FreshKeeper는 식재료의 남은 날짜를 자동으로 계산하고, 안전·주의·위험·오늘까지·만료 상태로 분류하여 사용자가 어떤 재료를 먼저 사용해야 하는지 쉽게 확인할 수 있도록 한다.

## 대상 사용자

- 냉장고 속 식재료를 체계적으로 관리하고 싶은 사용자
- 자취를 하며 식재료를 직접 관리하는 대학생
- 유통기한이 임박한 재료를 먼저 사용하고 싶은 사용자
- 보유 재료로 만들 수 있는 메뉴를 추천받고 싶은 사용자

## 기대 효과

- 유통기한이 임박한 식재료를 쉽게 확인할 수 있다.
- 냉장고 속 재료를 체계적으로 관리할 수 있다.
- 보유 재료를 활용한 메뉴 추천으로 음식물 낭비를 줄일 수 있다.
- 부족한 재료를 확인하여 장보기 계획에 도움을 줄 수 있다.
- Java의 클래스, 객체, ArrayList, 반복문, 예외처리, 날짜 계산 개념을 실제 프로젝트에 적용할 수 있다.

## 기술 스택

- 언어: Java
- 개발 도구: IntelliJ IDEA
- 버전 관리: Git, GitHub

## 주요 기능

| 기능 | 설명 |
|---|---|
| 식재료 등록 | 재료명, 카테고리, 수량, 보관 위치, 유통기한을 입력한다. |
| 식재료 조회 | 등록된 식재료 목록을 확인한다. |
| 식재료 삭제 | 선택한 식재료를 삭제한다. |
| 유통기한 계산 | 현재 날짜 기준으로 남은 날짜를 계산한다. |
| 위험도 분류 | 안전, 주의, 위험, 오늘까지, 만료 상태로 분류한다. |
| 메뉴 추천 | 보유 재료를 기반으로 만들 수 있는 메뉴를 추천한다. |
| 부족 재료 표시 | 추천 메뉴에 필요한 재료 중 없는 재료를 표시한다. |
