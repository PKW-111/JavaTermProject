## 5.6 발견된 오류 및 수정 내용

| 오류                  | 원인                            | 수정 내용                                                       | 결과 |
| ------------------- | ----------------------------- | ----------------------------------------------------------- | -- |

| 프로그램 재실행 시 데이터 초기화  | 메모리에서만 데이터를 관리함               | CSV 저장 및 불러오기 기능 구현                                         | 해결 |
| 검색 결과가 중복 표시됨       | 이름·카테고리·위치 검색 결과가 중복 추가됨      | 식재료 ID를 기준으로 중복 제거                                          | 해결 |
